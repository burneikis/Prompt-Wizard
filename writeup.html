Prompt Wizard

An Interactive RPG for Teaching AI Literacy

Technical Project Writeup

September 2024

Executive Summary

Prompt Wizard is an innovative web-based RPG designed to teach AI literacy to high school students through interactive gameplay. Students learn prompt engineering by crafting text-based spells to battle mystical creatures, with spell effectiveness determined by an AI evaluation system. The project successfully transforms abstract AI concepts into engaging, educational experiences suitable for classroom use.

Key Achievement: Created a fully functional educational game that makes AI literacy accessible to teenagers while maintaining educational rigor and safety standards.

Project Overview

Problem Statement

As AI systems become increasingly prevalent in education and daily life, students need foundational skills in AI literacy, particularly prompt engineering. Traditional educational approaches struggle to make these abstract concepts engaging and accessible for high school students.

Solution Approach

Prompt Wizard addresses this challenge by gamifying AI interaction through a fantasy RPG framework. Students learn by doingâ€”crafting spells (prompts) to battle creatures, receiving immediate feedback on their communication effectiveness, and naturally developing better AI interaction skills through gameplay.

Target Audience

  • Primary: High school students (Years 7-10, ages 12-16)
  • Secondary: Educators seeking AI literacy curriculum tools
  • Context: Classroom use or independent learning environments

Technical Architecture

Technology Stack

Frontend: React 18, CSS3 animations, responsive design

Backend: Node.js with Express.js framework

AI Integration: OpenAI API (gpt-4o-mini for evaluation, moderation API for safety)

Deployment: Vercel (frontend) + Railway (backend)

Data Storage: JSON files with planned database migration

Core Components

  • Prompt Evaluation Engine: AI-powered assessment of spell effectiveness
  • Content Moderation System: Safety filtering for all user inputs
  • Game State Manager: Creature health, player progression, score tracking
  • Educational Feedback System: Explanatory responses for learning reinforcement

Educational Design

Learning Objectives

Prompt Engineering

Students learn that specificity, clarity, and context significantly impact AI response quality through direct experimentation.

AI Understanding

Experience the probabilistic nature of AI systems and understand limitations through varied response patterns.

Responsible Use

Built-in moderation and reflection moments teach ethical AI interaction principles.

Critical Thinking

Students develop analytical skills by evaluating why certain prompts succeed or fail.

Pedagogical Approach

  • Experiential Learning: Learning through trial, error, and experimentation
  • Immediate Feedback: Real-time results show prompt effectiveness
  • Scaffolded Difficulty: Progressive complexity from simple to advanced creatures
  • Reflective Practice: Built-in analysis of successful strategies

Game Design & Mechanics

Creature System

Each creature teaches specific prompt engineering concepts through unique vulnerabilities:

  • Fire Dragon: Teaches elemental specificity and strategic thinking
  • Logic Sphinx: Emphasizes structured, logical prompt construction
  • Creative Phoenix: Encourages innovative and artistic expression
  • Shadow Wraith: Develops abstract thinking and metaphorical language
  • Ancient AI Oracle (Final Boss): Integrates all learned skills in multi-phase encounter

Core Gameplay Loop

  1. Encounter Selection: Player chooses creature/stage
  2. Spell Crafting: Text input for magical prompt
  3. Safety Check: Automatic content moderation
  4. AI Evaluation: Effectiveness assessment by OpenAI
  5. Battle Resolution: Damage/healing application
  6. Educational Feedback: Explanation of result

Progression & Engagement

  • Health System: Time pressure through decreasing player health
  • Spell History: Visual log of successful strategies for reference
  • Stage Progression: Unlockable content with replay capability
  • Scoring System: Points for prompt effectiveness and efficiency

Implementation Highlights

AI Evaluation System

The core innovation lies in sophisticated prompt evaluation:

const evaluationPrompt = `
Evaluate this spell for battling a ${creature.type}:
"${userPrompt}"

Consider:
- Clarity and specificity of the request
- Appropriateness for the target creature's vulnerabilities  
- Creative approach and innovation
- Technical feasibility of the described action

Respond with damage value (1-100) and brief explanation.
`;

Safety & Moderation

  • All user inputs processed through OpenAI's moderation API
  • Structured AI responses prevent inappropriate content generation
  • Educational messaging for rejected inputs
  • Comprehensive error handling for API failures

User Experience Features

  • Spell History Integration: Left-side panel showing successful spells for learning reinforcement
  • Healing Spell Detection: Natural language processing identifies healing intentions
  • Keyboard Shortcuts: Enter to submit, Shift+Enter for new lines
  • Stage Replay: Completed stages remain accessible for practice

Development Process

Agile Development Phases

Phase 1: Prototype

Basic React/Node.js setup, OpenAI integration, single creature encounter with prompt evaluation system.

Phase 2: Core Loop

Multiple creatures, progression system, health mechanics, spell history, scoring implementation.

Phase 3: Polish

UI/UX improvements, animations, safety features, stage system, world map integration.

Phase 4: Deploy

Production deployment, end-to-end testing, user feedback collection, performance optimization.

User Testing & Iteration

Testing with target demographic revealed key insights:

  • Positive: High engagement, natural experimentation with prompt styles, effective learning
  • Improvements: Added healing spell detection, spell history panel, keyboard shortcuts, stage replay
  • Ongoing: Content moderation fine-tuning, AI memory investigation

Technical Challenges & Solutions

AI Response Consistency

Challenge: Maintaining educational value while handling probabilistic AI responses.

Solution: Structured evaluation prompts with clear criteria and fallback mechanisms for unexpected responses.

Content Safety

Challenge: Ensuring age-appropriate content while maintaining engaging gameplay.

Solution: Multi-layer moderation with educational messaging for rejected inputs and curated response structures.

Educational Effectiveness

Challenge: Balancing game engagement with learning objectives.

Solution: Integrated feedback systems, spell history for reflection, and progressive difficulty that reinforces concepts.

Impact & Future Vision

Prompt Wizard successfully demonstrates that complex AI concepts can be made accessible through thoughtful game design. The project creates an intuitive bridge between abstract technical knowledge and practical skills, proving that AI literacy education can be both rigorous and engaging.

Key Success Metrics:

  • Students naturally develop better prompting strategies through gameplay
  • Educational objectives achieved without sacrificing engagement
  • Safe, moderated environment for AI experimentation
  • Scalable architecture supporting classroom deployment

Future Enhancements: Multiplayer features, teacher dashboard, advanced analytics, curriculum integration, and mobile applications will expand the project's educational impact.

As AI literacy becomes essential for digital citizenship, Prompt Wizard provides a foundation for responsible and effective human-AI interaction skills that will serve students throughout their academic and professional careers.
